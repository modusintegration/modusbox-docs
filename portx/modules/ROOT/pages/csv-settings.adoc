== Receive (Read)


[%header,cols="3s,7a"]

|Setting |Description

|Separates data in a document or message

Valid values:: xref:glossary#s[Standard PortX field format]

|Has Header Row

|Select this checkbox to include a header row for CSV Inbound documents
