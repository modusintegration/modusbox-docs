= Channels

A xref:glossary#c[*_Channel_*] provides a means of specifying the connection for a given partner. You can use the <<Channel Designer Page>> to visually craft and edit channels with pre-configured settings in PortX:


Type:: The channel type (Source or Target).

Document:: As created in xref:document-types[Document Types].

Document Map:: As created in xref:maps[Maps].

Endpoint:: As created in xref:endpoints[Endpoints].

=== Show All Channels
In some instances you may need to see all Channels available to a specific relationship.
To view all available Channels for a Relationship,
click the box next to *Show all channels visable to selected relationship* on the <<img-channels>>.

These additional Channels are available for use with this Relationship, and can be edited in the xref:transaction-designer.adoc[Transaction Designer].

== Channel Designer Page
To enter the Channel Designer page choose one of the following: 

* <<Add a New Channel>>
* <<Edit a Source Channel>>
* <<Edit a Target Channel>>

=== Add a New Channel

+
The <<img-channel>> appears.

=== Edit a Source Channel
. Select the Source Channel you want to edit from the <<img-channels>>.
+ 
The <<img-channel>> appears.

. You may select and edit:

* *Source Document*
* *Map*
* *Target Document*

Use configuration settings found in xref:document-types.adoc[Document Types] to configure your Source and Target Documents.

Use configuration settings found in xref:maps.adoc[Maps] to configure your map(s).

=== Edit a Target Channel
. Select the Target Channel you want to edit from the <<img-channels>>.
+
The <<img-channel>> appears.

. You may select and edit:

* *Source Document*
* *Map*
* *Endpoint*

Use configuration settings found in xref:document-types.adoc[Document Types] to configure your Source Documents.

Use configuration settings found in xref:maps.adoc[Maps] to configure your map.

Use configuration settings found in xref:endpoints.adoc[Endpoints] to configure your Endpoint.
