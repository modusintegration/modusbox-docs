= Maps

PortX facilitates EDI mapping with easy-to-use EDI mapping tools, the most important of which is a _map_. 

Configuring a map for a partner enables you to specify:

* Map name
* Source Document
* Target Document
* Mapping Type 
* Script

== Source Document, Target Document

When you click either the Source Document or the Target Document box, the Pick document selection page appears. You can select from available documents.

[%header,cols="3s,7a"]
|===
|Setting |Description

| Mapping Type

| Select a mapping type from the dropdown listbox.

Required?::
No

Valid values::

* 'Dataweave'
* 'XSLT'
* 'DataSonnet'

Default::

None
